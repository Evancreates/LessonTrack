# LessonTrack — 当前进度

已完成管理员与教师双角色的教学运营后台。

- 管理员：数据中心总览、课程管理、学生管理、教师管理、点名管理。
- 数据中心：学生、课时、今日计划课程、出勤率、教师课程进度、学生学习分布与课程运行看板；顶部入口为“刷新数据”。
- 课程：搜索/筛选、排序、拖拽列宽、启停确认、受报名关系保护的删除、全屏课程与学员进度编辑视图；课程计划时间用日历勾选维护，总课时由计划课次自动生成。
- 学生：Excel 导入、新增/编辑、报名课程维护、学习进度与出勤可视化详情；学生档案中的出勤/迟到/缺勤数字可点击查看明细。
- 教师：我的课程、我的学生、课堂点名；仅可访问本人课程及对应学生；教师管理支持新增教师角色的数据结构（姓名、账户名、密码）。
- 点名：管理员只查看已提交点名结果；教师端自动暂存，明确提交后才写入正式考勤并同步管理员看板；教师选中无计划课程日期时不显示名单；出勤和迟到均计入已上课时，只有缺勤不计入；教师可在“我的学生”修正已提交的历史考勤。
- 全站：恢复页面纵向滚动能力，并在长页面右下角提供“顶部 / 底部”快速滚动按钮。

npm run build
